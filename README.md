# DVT
Dynamic Visualization Toolkit
<br>
To install dependancies:
<br>
<ul>
<li>Install <a href= https://nodejs.org/> Node.js</a>, if you haven't done so already</li>
<li> Install Gulp:
<br>
```node
npm install gulp -g
```
</li>
<li>Navigate to lib subdirectory, and install dependancies:
<br>
```node
cd DVT/js/lib
npm install
bower install
```
</li>

<br>
To compile:
```node
cd DVT/js/lib
gulp 
```
